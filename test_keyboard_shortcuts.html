mdr Keyboard Shortcuts

File Operations

Ctrl+O / Cmd+O (Mac): Open file dialog
Ctrl+R / Cmd+R: Reload current file

View Controls

Ctrl+T / Cmd+T: Toggle Table of Contents visibility
Ctrl+P / Cmd+P: Pin/Unpin TOC sidebar
Ctrl+0 / Cmd+0: Reset font size to 100%
Ctrl+ / Cmd+: Increase font size
Ctrl- / Cmd-: Decrease font size

Theme Controls

Ctrl+Shift+L / Cmd+Shift+L: Cycle through light/dark/theme palettes
Ctrl+Shift+T / Cmd+Shift+T: Cycle through available themes

Navigation

Esc: Close TOC sidebar if open

Testing Instructions

Open the mdr application and try these keyboard shortcuts:

  1. Press Ctrl+O (or Cmd+O on Mac) to open a file
  2. Press Ctrl+T to toggle the Table of Contents
  3. Press Ctrl+P to pin/unpin the TOC
  4. Press Ctrl+ and Ctrl- to change font size
  5. Press Ctrl+0 to reset font size
  6. Press Ctrl+Shift+L to cycle through palettes
  7. Press Esc to close the TOC if it's open

You should see status messages appear in the status bar confirming the actions.
